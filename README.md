# Genius Tutorial Website

A modern landing page and coaching institute web app built with Vite, React, Tailwind CSS, and Radix UI components. This project is inspired by a coaching institute design system and includes a clean multi-page layout for courses, faculty, results, contact, and app download sections.

## Features

- Responsive React app with client-side routing via `react-router`
- Tailwind CSS styling and theming
- Reusable UI components from Radix UI
- Animated interactions and modern layout design
- Pages included:
  - Home
  - About
  - Courses
  - Faculty
  - Results
  - Contact
  - Download App
- Built as a static SPA with Vite

## Tech Stack

- `vite`
- `react`
- `react-router`
- `tailwindcss`
- `@radix-ui/react-*`
- `@mui/material` + icons
- `react-hook-form`
- `recharts`
- `lucide-react`
- `react-slick`
- `date-fns`

## Project Structure

- `src/main.tsx` — app entry point
- `src/app/App.tsx` — root app component
- `src/app/routes.tsx` — router configuration
- `src/app/layouts/MainLayout.tsx` — main site layout
- `src/app/pages/` — page components
- `src/app/components/ui/` — shared UI primitives
- `src/styles/` — global and theme CSS
- `public/` — static assets
